## Структура документа

Каждый кейс содержит:
1. **Резюме кандидата** — полные данные из candidate_json
2. **Вакансия** — полные данные из vacancy_json
3. **Judge (эталон)** — оценка gpt-4.1 с обоснованием
4. **Prompt A** — результат с детализацией по 4 категориям
5. **Prompt B** — результат с детализацией по 4 категориям

**Шкала оценки:**
- role_score: 0–30 баллов
- skills_score: 0–35 баллов
- experience_score: 0–20 баллов
- conditions_score: 0–15 баллов
- **Итого:** 0–100 баллов
